欢迎来到我的新博客！

这是一个使用 **Astro** 构建的静态网站。它具有以下特点：

1. **极速加载**：Astro 默认不发送 JavaScript 到客户端。
2. **Markdown 支持**：我可以专注于写作，只需编写 `.md` 文件即可发布文章。
3. **响应式设计**：使用 Tailwind CSS 打造，完美适配移动端。
4. **暗黑模式**：保护你的眼睛。
